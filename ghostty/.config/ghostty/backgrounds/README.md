## Adding New Backgrounds

1. Download or add your image to this folder
2. Update `ghostty/config`:
   ```
   background-image = /Users/rwarne/dotfiles/ghostty/backgrounds/your-image.jpg
   ```
3. Adjust opacity if needed:
   ```
   background-opacity = 0.9  # Range: 0.0 (transparent) to 1.0 (opaque)
   ```
4. Restart Ghostty to see changes

## Recommended Settings

- **Opacity:** 0.85-0.95 for good text readability
- **Format:** JPG or PNG
- **Theme pairing:** Choose terminal colors that contrast well with your background
